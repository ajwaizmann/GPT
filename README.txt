GPT Capstone Project

This program currently only has the capability to act as a GPT chat bot. It uses an openai API key to send queries.

1. You must install openai bindings before running. "pip install openai"
2. You must install decouple. "pip install python-decouple"
3. Create a file called .env and add a varibale called APIKEY. "APIKEY = ..."
4. Run main.py
5. Entrer "exit" to terminate program.
